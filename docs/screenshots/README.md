# Screenshots

Place redacted deployment screenshots in this folder.

## Recommended Files

- `aws-ec2-instance.png` - EC2 instance state and checks passed
- `cloudwatch-billing-alarm.png` - billing alarm in OK state
- `frontend-homepage.png` - BrewHaven landing page
- `menu-page.png` - product menu grid
- `api-products-ok.png` - browser or terminal proof of `200 OK`

## Redaction Notes

- Crop out browser tabs and address bar if they show anything private
- Hide AWS account numbers if visible
- Keep the proof readable, but minimal
- Never include credentials, keys, or tokens
